Adhan Audio Files
==================

This directory should contain sample Adhan audio files for notifications.

You can add your own adhan audio files (MP3 format) here.

Suggested files:
- adhan1.mp3 (Traditional adhan)
- adhan2.mp3 (Alternative adhan)

File requirements:
- Format: MP3
- Duration: 2-5 minutes recommended
- File size: Keep under 5MB per file for best performance
- Sample rate: 44.1kHz or 48kHz

The app will use these files for Adhan notifications when enabled in settings.

Free Islamic audio resources:
- https://www.islamicfinder.org/
- Various Islamic websites offering royalty-free adhan recordings

Note: Make sure you have the right to use any audio files you add.
