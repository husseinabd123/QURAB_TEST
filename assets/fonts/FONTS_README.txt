Cairo Font Files Required
==========================

Please download the Cairo font family from Google Fonts and place the following files in this directory:

Required files:
- Cairo-Regular.ttf
- Cairo-Medium.ttf
- Cairo-SemiBold.ttf
- Cairo-Bold.ttf

Download Link:
https://fonts.google.com/specimen/Cairo

Instructions:
1. Visit the link above
2. Click "Download family"
3. Extract the downloaded zip file
4. Copy the required .ttf files to this directory (/workspace/assets/fonts/)
5. Make sure the filenames match exactly as listed above

The app will not build without these font files.
